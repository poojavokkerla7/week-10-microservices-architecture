
# 2. Eureka Server 



This is the Service Registry for the Microservices Architecture project using Spring Cloud Netflix Eureka.

---

## Overview
Eureka Server is responsible for service discovery. All microservices register here and communicate using service names instead of hardcoded URLs.

---

## Tech Stack
- Java
- Spring Boot
- Spring Cloud Netflix Eureka Server
- Maven

---

## Features
- Service registration
- Service discovery
- Health monitoring of services
- Dashboard for registered services

---

## How to Run

```bash
mvn spring-boot:run
