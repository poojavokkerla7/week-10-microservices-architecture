# User Service

This is the User Microservice in a Spring Cloud Microservices Architecture.

---

## Overview
The User Service handles all user-related operations such as:
- Creating users
- Fetching user details
- Updating user data
- Deleting users

It registers itself with Eureka Server and is accessed via API Gateway.

---

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Spring Web
- Spring Cloud Netflix Eureka Client
- MySQL / H2 Database
- Maven

---

## Features
- REST APIs for user management
- Service registration with Eureka
- Database integration (JPA/Hibernate)
- Exception handling
- Layered architecture (Controller, Service, Repository)
